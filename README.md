## 설치 및 사용방법

### 설치

```bash
git clone https://github.com/RyuJun/fluent-dashboard.git
```

```bash
cd fluent-dashboard
```

```bash
yarn install
```

### 실행

> localhost:6006

```bash
yarn dev-server
```
